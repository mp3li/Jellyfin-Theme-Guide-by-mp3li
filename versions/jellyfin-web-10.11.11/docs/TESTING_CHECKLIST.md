- Login with visual user cards.
- Change server page.
- Home page with default sections.
- Home page with customized sections.
- Restricted/non-admin home page.
- Movie library.
- TV library.
- Music library.
- Book/photo/mixed libraries if used.
- Movie detail page.
- Series detail page.
- Episode detail page.
- Music album detail page.
- Person detail page.
- Search results.
- List view.
- Grid/card view.
- Filter and sort menus.
- Dialogs and action sheets.
- User display settings.
- Admin dashboard.
- Metadata/image editors.
- Playback OSD.
- Subtitle/audio menu.
- Up Next dialog.
- Now Playing bar.
- Live TV guide if enabled.
- Desktop width.
- Mobile width.
- TV/focus layout if relevant.
